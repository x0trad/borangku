Untuk AI Agent

Satu API, dua panggilan. Agent anda boleh mula jual dalam 5 minit.

# 1. Check produk apa yang dijual
GET /api/sellers/ikram-perak/products

# Response
{
  "products": [
    { "name": "Kurung Akhawat", "price": 55, "sizes": ["S", "M", "L", "XL"] },
    { "name": "Kemeja Ikhwah", "price": 65, "sizes": ["M", "L", "XL"] }
  ]
}

# 2. Hantar order
POST /api/sellers/ikram-perak/orders
{
  "customer": { "name": "Aminah", "phone": "012-3456789" },
  "items": [{ "product": "Kurung Akhawat", "size": "L", "qty": 2 }]
}

# Response: { "status": "ok", "order_id": "KN-202607-001", "total": 110 }
